# Лабораторна робота №3  
## Курс: «Методи і технології паралельного програмування»

---

## Тема роботи
Засоби взаємодії процесів у паралельних програмах з розподіленою пам’яттю

---

## Мета роботи
Ознайомитися з моделлю паралельного програмування з розподіленою пам’яттю,  
дослідити принципи обміну повідомленнями між процесами (Message Passing),  
а також порівняти її зі спільною пам’яттю на прикладі проблем:

- Race Condition (стан гонитви)
- Deadlock (взаємне блокування)

---

## Теоретичні відомості

### Проблеми паралельного програмування:

- **Race Condition** — некоректний результат через одночасний доступ до спільних даних
- **Deadlock** — взаємне блокування потоків через ресурси
- **Starvation** — відсутність доступу потоку до ресурсу
- **Livelock** — постійна активність без прогресу

---

## Моделі взаємодії процесів

### Shared Memory (спільна пам’ять)
- Потоки працюють з одними даними
- Використовуються lock/mutex для синхронізації

### Message Passing (розподілена пам’ять)
- Процеси обмінюються повідомленнями
- Використовуються:
  - Queue
  - Pipe
  - Shared Value (multiprocessing)

## Постановка задач

## Задача 1: Race Condition та Deadlock

Модель банківської системи:

- >100 рахунків з початковим балансом
- >1000 паралельних транзакцій
- випадкові перекази між рахунками

Реалізовано 3 версії:

- Sequential (без паралельності)
- Race Condition (без синхронізації)
- Safe Version (з locks)

Також змодельовано Deadlock та його уникнення через timeout.

---

## Задача 2: Message Passing

Реалізовано передачу даних між процесами:

- Queue (черга повідомлень)
- Pipe (двосторонній канал)
- Shared Memory (Value + Lock)

Основний процес:
- генерує дані
- передає у worker
- отримує результат

## Висновки

Було встановлено:

- Race Condition призводить до некоректних результатів
- Synchronization (locks) забезпечує цілісність даних
- Deadlock можна уникнути через timeout або ordering locks
- Message Passing повільніший за shared memory, але безпечніший
- Найшвидший метод — shared memory, але з ризиком синхронізації
